*Rogue Maneuver (Spellthief)*

# Scan

**At-will, Swift, Ranged**

You analyze a subject's magical signature, learning if it possesses magical abilities. Roll an Investigation check; you learn which spells subject has available to cast based on their rank (DC 10 + 5 per rank).
